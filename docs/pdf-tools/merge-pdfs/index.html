Why Merge PDFs in the Browser?

Most online PDF mergers require you to upload your files to a remote server. That means your documents — contracts, reports, medical records, tax forms — are transmitted over the internet and stored temporarily (sometimes permanently) on someone else's infrastructure. This tool uses pdf-lib, a pure-JavaScript PDF library, to combine your files entirely within your browser tab.

  • No file upload — files are read directly from your device's memory
  • No server processing delays — merging happens at JavaScript speed
  • No file size limits enforced by a server — your device's RAM is the only constraint
  • 100% free, no account required, no watermarks added to the output

Frequently Asked Questions

How do I merge PDF files?

Click the upload zone or drag and drop multiple PDF files onto the page. Each file appears as a row showing the filename and page count. Drag rows up or down to set the order you want. When ready, click Merge PDFs to combine them and download the result instantly.

Is there a file size limit?

There is no server-imposed file size limit because all processing happens in your browser. The practical limit is your device's available memory. Most modern computers can merge dozens of PDFs totalling hundreds of megabytes without issue.

Are my PDFs kept private?

Yes — completely. Your PDF files are read locally on your device and are never uploaded to any server. All merging happens in your browser tab's memory and the final file downloads directly to your computer. Nothing is transmitted or stored anywhere.

Can I reorder pages before merging?

You can reorder the files by dragging rows in the list. All pages within each PDF are included in their original order. To reorder individual pages across documents, use the Extract Pages tool to create PDFs containing only the pages you need, then merge them here in the desired order.
